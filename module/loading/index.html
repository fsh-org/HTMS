module "loading"

Loading Module

This module lets you add progress vars

Add a <pb></pb> tag to add straight progress bar

You can add the following in style to stylize the bar

  • --width current progress
  • --color covered part color
  • --background non covered color
  • --radius the bars radius
  • --txtcolor color of the text
  • --txtpos where the text is (left, center, right)
  • --txtfont font used for text
  • --txt:none to remove the text
