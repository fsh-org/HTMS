BackToTop Module

This module adds a back to top button to return to the top of the page if you have scrolled down a lot

To customize add a <btt></bbt> element and add atributes to change it

<btt attribute="value"></bbt>

Avaible attributes:

  • color - arrow color
  • background - arrow background (color, image...)
  • border - the border
  • radius - border radius
  • place - left/right were to put the arrow
